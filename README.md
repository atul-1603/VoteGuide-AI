# 🗳️ VoteGuide — AI Election Assistant

VoteGuide is a production-grade, fully modular AI-powered election assistant web application. 
The app helps citizens understand the election process, timelines, voter registration, 
and their rights — through an intelligent conversational interface and structured informational 
sections.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js 14 (App Router, TypeScript)
- **Styling**: Tailwind CSS + shadcn/ui
- **AI**: Google Gemini 2.0 Flash
- **Auth**: Firebase Auth
- **Database**: Firebase Firestore
